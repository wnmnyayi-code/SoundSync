# SoundSync Mobile Apps

This directory contains the mobile application installers for SoundSync.

## Files

- `SoundSync-Android.apk` - Android application package (38.5 MB)
- `SoundSync-iOS.ipa` - iOS application package (45.2 MB)

## Installation Instructions

### Android (APK)

1. Download `SoundSync-Android.apk` to your Android device
2. Enable "Install from Unknown Sources" in Settings > Security
3. Open the downloaded APK file
4. Follow the installation prompts
5. Launch SoundSync from your app drawer

**Requirements:**
- Android 8.0 (Oreo) or later
- 100 MB free storage space
- Internet connection for streaming

### iOS (IPA)

**Note:** IPA files require a developer certificate or jailbroken device for installation.

**For App Store Installation:**
1. Visit the App Store on your iOS device
2. Search for "SoundSync"
3. Tap "Get" to download and install

**For Direct Installation (Developers):**
1. Download `SoundSync-iOS.ipa`
2. Use Xcode or a tool like Cydia Impactor
3. Sign the app with your developer certificate
4. Install to your device

**Requirements:**
- iOS 15.0 or later
- iPhone, iPad, or iPod touch
- 150 MB free storage space
- Internet connection for streaming
