🧠 Updated SoundSync System Specification
🌍 Availability

South Africa only (geo-restricted for trial/launch)

Local currency: ZAR (Rands)

VAT = 15% applied automatically to all transactions

SARS tax reporting & deduction automated through admin dashboard

👥 User Types (Subscription Model)
Tier	Description	Account Options
Basic (Free)	Access one role	Fan or Artist or Merchant or Influencer
Standard (Paid)	Access two roles	Choose any 2 of 4 roles
Premium (Paid)	Access all roles	Full access to all 4 roles
Roles
🎤 Artist

Must declare status: Independent / Signed / Registered

Must provide SARS clearance certificate

Must disclose SAMBRO registration (or apply for it via SoundSync for a fee)

Must declare sampled music

Can create & host live playlists (streamed from owned local files)

Can buy extra playlist slots

Can upload high-res profile pictures

Can sell merchandise, digital music, or sound tools

Delivery radius for physical goods must be stated

Can hire influencers for promotion (price negotiated privately)

Can pay SoundSync for professional mixing/mastering if audio quality is low

Withdrawals only available after reaching R200 minimum

🎧 Fan

Can join live listen parties (paid RSVP)

Can interact and follow artists, artists can follow back

Can create playlists using licensed music (royalty fee applies via SAMBRO)

Can buy merchandise or digital products

Can upgrade to creator/influencer/merchant on request (pending admin approval)

💼 Merchant

Can sell physical/digital items (mixers, sound tools, effects, plugins, etc.)

Must provide SARS clearance certificate

Can set delivery radius

Doesn’t need to be an artist/fan to sell

VAT automatically calculated

📣 Influencer

Gets paid only when their referral buys or uses coins

Payment = percentage-based commission from those user transactions

Commission visible on dashboard

Withdrawals available after R100 minimum

Also eligible for SoundSync referral bonuses based on verified subscribers they bring

💰 Financial & Payment Logic
Example: Fan spends R100 on artist promoted by influencer
Party	Percentage	Amount (R)	Notes
Artist	60%	R60	Main revenue earner
Influencer	10%	R10	Only if referral is verified
SoundSync	15%	R15	Platform fee for operations
SARS (VAT)	15%	R15	Automatically remitted

Total Revenue = R100 (VAT included)

Withdrawal Limits

Artist: min R200

Influencer: min R100

Merchant: min R500

Coin Packages

R10 = 200 coins

R25 = 500 coins

R50 = 1000 coins
(Coins usable for RSVPs, tips, merchandise, and digital purchases)

📊 Admin & Audit

Full automatic VAT and commission tracking

Financial audit tool available (fee-based on request)

Downloadable tax return reports

SARS integration for direct remittance

Mix/Master tool embedded for creator music enhancement

Audit requests charged at an admin-set rate
