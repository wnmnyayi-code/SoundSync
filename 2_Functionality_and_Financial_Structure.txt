2. FINANCIAL STRUCTURE

2.1 REVENUE DISTRIBUTION MODEL

Standard Transaction (R100 spent by fan):
- Artist: R60.00 (60%)
- Influencer: R10.00 (10%) - if referred
- Platform: R15.00 (15%)
- VAT: R0.00 (currently disabled)
- Total: R100.00

Without Influencer Referral:
- Artist: R60.00 (60%)
- Platform: R25.00 (25%)
- VAT: R0.00 (currently disabled)
- Remaining: R15.00 (redistributed to platform)

2.2 COIN ECONOMY

Official Coin Packages:
- R10 = 200 coins (R0.05 per coin)
- R25 = 500 coins (R0.05 per coin)
- R50 = 1000 coins (R0.05 per coin)

Custom Amounts:
- Minimum: R10
- Maximum: R10,000 per transaction
- Same rate: R0.05 per coin

Coin Usage:
- Tip artists (any amount)
- RSVP for live sessions (set by artist)
- Purchase tracks (set by artist)
- Buy merchandise (set by merchant)
- Access premium content

2.3 SUBSCRIPTION PRICING

BASIC TIER (Free):
- Access: 1 user role
- Features: Basic platform features
- Support: Community support
- Analytics: Basic stats
- Cost: R0/month

STANDARD TIER (R99/month):
- Access: 2 user roles
- Features: Advanced features
- Support: Priority support
- Analytics: Enhanced analytics
- Early Access: New features
- Cost: R99/month

PREMIUM TIER (R199/month):
- Access: All 4 user roles
- Features: Premium features
- Support: VIP support
- Analytics: Advanced analytics
- Early Access: Beta features
- Bonuses: Performance rewards
- Cost: R199/month

2.4 MERCHANT COMMISSION STRUCTURE

Sales Revenue Split:
- Merchant: 90% of sale price
- Platform: 10% commission

Example: R1000 Product Sale
- Merchant receives: R900
- Platform receives: R100

No Additional Fees:
- No listing fees
- No monthly fees
- No transaction fees (beyond 10%)
- No withdrawal fees

2.5 WITHDRAWAL SYSTEM

Minimum Thresholds:
- Artists: R1000
- Influencers: R1000
- Merchants: R1000

Processing Timeline:
- Withdrawal request: Instant
- Platform processing: 24-48 hours
- Bank transfer: 1-3 business days
- Total time: 2-5 business days

Withdrawal Methods:
- South African bank transfer (primary)
- International wire transfer
- PayPal (coming soon)

Withdrawal Fees:
- South African banks: Free
- International transfer: Bank fees apply
- PayPal: 2.5% + R5 (when available)

2.6 PAYMENT PROCESSING

Payment Gateway: Stripe
- PCI DSS compliant
- Secure card processing
- Multiple currency support
- Fraud detection
- 3D Secure authentication

Accepted Payment Methods:
- Credit cards (Visa, Mastercard, Amex)
- Debit cards
- Bank transfers
- Mobile money (coming soon)

Transaction Security:
- SSL encryption
- Tokenized card storage
- No card details stored on platform
- Automated fraud detection

2.7 TAX STRUCTURE (CURRENTLY DISABLED)

VAT Status:
- VAT Rate: 15% (South Africa standard)
- Current Status: DISABLED
- Implementation: Future phase

When VAT is enabled:
- Will be added to all transactions
- Collected by platform
- Remitted to SARS
- Separate line item on invoices

SARS Compliance:
- Artists: Optional SAMBRO number
- Merchants: Mandatory SARS clearance certificate
- Platform: Registered VAT vendor (when enabled)
