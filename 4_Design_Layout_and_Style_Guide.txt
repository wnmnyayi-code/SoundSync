===========================================
SOUNDSYNC - DESIGN, LAYOUT & STYLE GUIDE
===========================================

1. DESIGN PHILOSOPHY

SoundSync's design embodies a modern, music-focused aesthetic that balances professionalism with creativity. The platform uses a dark theme with vibrant purple and pink accents to create an immersive, premium music experience.

Core Design Principles:
- Music-first visual hierarchy
- Clean, uncluttered interfaces
- Smooth animations and transitions
- Accessibility and readability
- Mobile-responsive design
- Performance-optimized

===========================================

- Hover: Slight opacity reduction

Outline Button:
- Class: border-border text-foreground hover:bg-card

- Hover: Card background
- Background: Transparent

Ghost Button:
- Class: hover:bg-accent/10
- Background: Transparent
- Hover: Accent with 10% opacity
- No border

Button Sizes:
- sm: Small buttons
- md: Default size
- lg: Large buttons

5.2 CARDS

Standard Card:
- Class: bg-card border-border shadow-card
- Background: Card background color
- Border: Border color
- Shadow: Card shadow effect

6. VISUAL EFFECTS

6.1 SHADOWS

Card Shadow:
- box-shadow: 0 10px 30px -10px hsl(270 70% 20% / 0.5)
- Used for: Cards, elevated elements

Glow Shadow:
- box-shadow: 0 0 40px hsl(270 70% 60% / 0.3)
- Used for: Buttons, featured cards, hover states

6.2 ANIMATIONS

Transitions:
- transition-all: All properties
- transition-colors: Color changes
- transition-transform: Transform changes
- Duration: 200-300ms (default)

Hover Effects:
- Scale: hover:scale-110 (10% larger)
- Opacity: hover:opacity-90 (90% opacity)
- Shadow: hover:shadow-glow (add glow)
- Border: hover:border-primary/50 (primary border)

Loading States:
- Spinner animation
- Pulse effect: animate-pulse
- Skeleton loaders

6.3 BACKDROP EFFECTS

Backdrop Blur:
- backdrop-filter: blur(20px)
- Used for: Navigation, modals, overlays

Background Blend:
- mix-blend-mode: multiply
- Used for: Background overlays
- Creates depth and atmosphere

9. RESPONSIVE DESIGN

9.1 MOBILE FIRST

Approach:
- Design for mobile first
- Progressive enhancement
- Touch-friendly targets (min 44px)
- Readable text sizes (min 16px)

Mobile Optimizations:
- Single column layouts
- Stacked navigation
- Full-width buttons
- Simplified interactions

9.2 TABLET ADAPTATIONS

Layout Changes:
- 2-column grids
- Side-by-side elements
- Larger touch targets
- More whitespace

Navigation:
- Horizontal navigation
- Dropdown menus
- Tablet-optimized spacing

9.3 DESKTOP ENHANCEMENTS

Layout:
- Multi-column grids (3-4 columns)
- Sidebar navigation
- Hover effects
- Larger content areas

Interactions:
- Mouse hover states
- Keyboard navigation
- Tooltips
- Context menus

10. ACCESSIBILITY

10.1 COLOR CONTRAST

Text Contrast:
- Foreground on background: 15:1 (AAA)
- Muted text on background: 4.5:1 (AA)
- Primary on background: 7:1 (AAA)

Interactive Elements:
- Buttons: 4.5:1 minimum
- Links: 4.5:1 minimum
- Form inputs: 4.5:1 minimum

10.2 FOCUS STATES

Keyboard Focus:
- Ring: ring-2 ring-primary
- Offset: ring-offset-2
- Visible on all interactive elements

Focus Indicators:
- Buttons: Outline ring
- Links: Underline + ring
- Inputs: Border + ring

10.3 ARIA LABELS

Screen Reader Support:
- Proper heading hierarchy
- Alt text for images
- ARIA labels for icons
- Descriptive link text

Semantic HTML:
- nav, header, main, footer
- article, section, aside
- button, a, input elements
- Proper form labels
